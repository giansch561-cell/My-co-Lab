# Definição do Produto — My-co-Lab

## Nome atual

My-co-Lab Core Prototype V0

---

## Estado atual

Protótipo funcional de monitorização ambiental.

Ainda não é um sistema completo de cultivo automatizado.

---

## Descrição atual

Sistema offline de monitorização ambiental para cultivo de cogumelos com:

- sensores
- Raspberry Pi Pico
- Raspberry Pi 4
- LCD local
- logging CSV
- backup automático USB
- GitHub
- operação offline

---

## Objetivo do Core

Criar uma base robusta antes de adicionar automação.

O Core deve ser:

- estável
- modular
- recuperável
- simples
- bem documentado
