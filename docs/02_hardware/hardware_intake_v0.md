## Objective

Validate newly received hardware before integration.

## Components

- humidifier

## Validation Checklist

### Visual Inspection

- damage
- connectors
- cable quality

### Electrical

- voltage confirmation
- power stability
- heat generation

### Functional Test

- isolated power-on test
- fan airflow
- humidifier mist generation
- vibration/noise observation

## Rules

- no Raspberry Pi integration
- no box integration
- no software changes
- test components individually
