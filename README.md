# My-co-Lab 🍄

Sistema modular de cultivo inteligente de cogumelos usando:

- Raspberry Pi 4
- Raspberry Pi Pico
- sensores
- automação
- timelapse
- documentação
- operação offline

---

## Filosofia

Learn by building.

Document everything.

Improve continuously.

---

## Objetivo

Criar um ecossistema modular de cultivo inteligente que combine:

- cultivo
- tecnologia
- automação
- documentação
- conteúdo
- produto futuro

O objetivo não é dinheiro rápido.

O objetivo é construir algo real, útil e sustentável.
